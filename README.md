# Simply Expression Language

#### 介绍
简易表达式，旨在解决判断、计算、取值、结构转换这些逻辑简单但编码繁复的工作

#### 特性
支持层级结构：快速操作、转换JSON或各类POJO等<br>
安全取值：告别频繁繁琐的空判断


#### 引入配置
```xml
<dependency>
    <groupId>org.fanjr.simplify</groupId>
    <artifactId>simplify-el</artifactId>
    <version>1.0.5</version>
</dependency>
```
